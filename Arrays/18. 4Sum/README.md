Given an array `nums` of `n` integers, return **all the unique quadruplets** `[nums[a], nums[b], nums[c], nums[d]]` such that:

- `0 <= a, b, c, d < n`  
- `a, b, c, d` are **distinct**  
- `nums[a] + nums[b] + nums[c] + nums[d] == target`

You may return the answer in **any order**.
